\subsection{Detailed Description}
Provide iif\+\_\+assert and iif\+\_\+assume support for system assume and assert macros. 

For each valid test, iif\+\_\+assume and iif\+\_\+assert should be called only once. Otherwise, the test is regarded as an invalid test.
